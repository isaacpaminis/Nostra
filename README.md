Nostra – Responsive E-commerce Website

Nostra is a responsive e-commerce website developed using HTML, CSS, and JavaScript.

This project focuses on building a clean user interface and implementing interactive features such as product search and filter functionality using JavaScript DOM manipulation.

Features

Fully responsive design (Mobile, Tablet, Desktop)

Product search functionality on the Collections page

Product filtering based on categories

Mobile-friendly navigation menu

Multi-page website structure (Home, Collections, Contact)

Technologies Used

HTML

CSS

JavaScript

Tailwind CSS
